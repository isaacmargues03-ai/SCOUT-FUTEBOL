# **App Name**: Dependency Tracker

## Core Features:

- Dependency List: Display a list of all dependencies in the project's package.json.
- Vulnerability Scan: Scan dependencies for known vulnerabilities using the npm API.
- Outdated Check: Check for outdated dependencies using npm outdated and display the results.
- License Checker: Verify that all the licenses for the various npm packages are compatible with each other.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to convey trust and stability in dependency management.
- Background color: Light gray (#F0F2F5), a slightly desaturated version of the primary, for a clean interface.
- Accent color: Soft orange (#D96A2F) for alerts related to security or outdated packages.
- Font pairing: 'Inter' for both headings and body text (sans-serif) for clean and modern readability.
- Code font: 'Source Code Pro' for displaying the project dependencies in package.json.
- Simple, clear icons to represent dependency status (up-to-date, outdated, vulnerable).
- Clean and structured layout with clear visual hierarchy to display dependencies and their status.
- Subtle transitions when updating dependency statuses.
